📌 Sobre o Projeto

Este sistema foi desenvolvido para automatizar o sorteio de partidas 5x5 de Counter-Strike (CS) dentro de servidores de Discord.

A ferramenta organiza 10 jogadores, considera o rank/nível de cada jogador e realiza uma divisão equilibrada em dois times, promovendo partidas mais justas e competitivas.

Também inclui a funcionalidade de sorteio de mapas, tornando a experiência ainda mais completa.

⚙️ Funcionalidades
✔️ Seleção de 10 jogadores
✔️ Balanceamento automático por rank
✔️ Divisão em dois times equilibrados (5x5)
✔️ Sorteio de mapas
✔️ Interface simples e responsiva
✔️ Execução rápida e automática
